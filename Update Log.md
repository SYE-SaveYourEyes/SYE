# SYE Update Log

---

## Version 1.0

**Released: 25/03/26**

Initial release of SYE — Save Your Eyes.

---

## What’s Included

### Quick App Blocks

* Five customizable preset slots
* One-time app assignment using Screen Time picker
* Durations: 30m, 1h, 2h, 4h, 8h
* Multiple simultaneous blocks
* Persistent timers across restarts
* Active block display with cancel option
* Custom shield screen with block timer
* Fully renameable and reassignable presets

---

### Soft Interruptions

* Track up to 10 apps
* Pause screen with motivational prompt
* “Yes I need it” → temporary access (~30s)
* “Go back” → exit app
* Disabled by default
* Inline app renaming

---

### Expected Phone Use

* Automatic mode using Screen Time data
* Manual mode for instant projection
* Lifetime usage calculation in years/months
* Based on:

  * 8h sleep
  * 8h work/school
  * Daily life activities
* Approx. 5.3 free hours/day baseline
* Based on 73-year life expectancy
* Full formula explanation included

---

### Notifications

* Daily Screen Time threshold alerts (1–4h)
* Fallback notification system
* Disabled until permission granted

---

### General

* Screen Time permission gate on first launch
* All features default OFF
* App Group shared storage
* Separate ManagedSettingsStore for features
* App name: SYE
* Privacy-first:

  * No servers
  * No analytics
  * No data collection

---

## Known Issues (v1.0)

* First preset tap after restart may show blank screen
* Block timers may vary by 1–5 minutes (system limitation)
* Automatic mode may take up to 24h to populate
